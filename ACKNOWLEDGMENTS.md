# Acknowledgments

DocFlow AI is built entirely on top of excellent open-source projects. Thanks
to the maintainers and communities behind:

- [FastAPI](https://fastapi.tiangolo.com/) — web framework
- [Ollama](https://ollama.com/) — local LLM runtime
- [ChromaDB](https://www.trychroma.com/) — local vector database
- [sentence-transformers](https://www.sbert.net/) — local embedding models
- [Tesseract OCR](https://github.com/tesseract-ocr/tesseract) — optical character recognition
- [Apache Tika](https://tika.apache.org/) — generic document parsing
- [poppler](https://poppler.freedesktop.org/) — PDF rendering utilities
- [SQLAlchemy](https://www.sqlalchemy.org/) — ORM and database toolkit
- [python-docx](https://python-docx.readthedocs.io/) and [openpyxl](https://openpyxl.readthedocs.io/) — Office document parsing
- [Meta Llama](https://ai.meta.com/llama/) models, served locally via Ollama

This project would not be possible without the broader open-source AI and
Python ecosystem.
